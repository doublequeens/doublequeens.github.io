# John Q Logo 调整经验与设计规范总结

本文档总结了 "John Q" 标志在基于纯 SVG 代码手工还原和微调过程中的所有关键经验、几何比例规则以及细节特征。整个调整过程遵循了严格的数学比例和像素级对齐要求。

## 全局设计原则与规范

- **基线统一 (Baseline)**：所有字母 (`J`, `o`, `h`, `n`, `Q`) 的最底端严格驻扎在同一水平基线 (y=240) 上。
- **双重顶高系统**：
  - **大写/长高字母**：`J`、`h` (左侧竖线)、`Q` 共用最高顶线 (y=30)。
  - **小写/短高字母**：`o`、`h` (右侧拱形)、`n` (左侧竖线与最高点) 共用中段顶线 (y=100)。
- **字间距等距 (Spacing)**：`J` 与 `o`、`o` 与 `h`、`h` 与 `n` 之间的水平间距全部通过精确的 X 坐标平移，统一调整至严格相等的 **40px**。
- **线条特征 (Stroke)**：统一采用粗线条 (`stroke-width="25"`)，端点使用圆角闭合 (`stroke-linecap="round"`)，以体现现代与复古结合的质感。
- **颜色 (Color)**：品牌色最终确定为纯正的橄榄绿 (**Olive**, `#808000`)。

## 各字母特征与调整详解

### 1. 字母 `J`
- **基础结构**：一条直立的长竖线配合底部的左侧弯月钩。
- **高度特征**：整个单词中的最高点，左上方长线直达最高顶线，底部到达基线。
- **底部弧度**：底部的勾形是一个 **90°的四分之一圆弧**，向左侧平滑过度至基准线的绝对底端。

### 2. 字母 `o`
- **比例与高度**：整体高度约为 `J` 的 2/3。顶部恰好与长竖线的中上部齐平（即中段顶线），底部贴合基线。
- **形态特征**：接近绝对正圆的几何框形 (`r=70`)。
- **关键细节（缺口）**：并非闭合圆，在原本的 **左下角（6点钟到9点钟方向）** 有一个明显的定向缺口，开口方向精准。

### 3. 字母 `h`
- **非对称高度设计**：左侧竖直线是从头贯穿到脚的最高点级别（等同于 J），没有任何倾斜或弯曲。
- **分离式拱门**：这是 `h` 最核心的特征。右侧的拱门与左侧竖线**不相接**，中间刻意留白产生缝隙断层。
- **拱形曲度与高度**：曾经历过 90° 到半圆的多次推敲，最终定稿为 **135° 的圆弧**，以完美契合从切线自然下坠的重心。同时，**右侧拱形的最高点必须严格与身边 `o` 字母的最高点保持一致**。

### 4. 字母 `n`
- **高度一致性**：`n` 的左侧竖线不再是贯彻全局的高线，而是**严格等同于 `o` 的高度**（中段顶线）。
- **几何级直角拱门**：右侧同样是不相接的断层拱门。但这边的拱门是一个绝对意义上的 **90°四分之一圆弧**。
- **黄金长度**：右侧圆弧落下后，衔接的是一根绝对垂直的短竖线，这根短竖线的长度**非常严格地等于左侧主体竖线长度的一半**，踩在基线上。

### 5. 字母 `Q`
- **基准身型**：和 `J` 与 `h` 的长竖线等高，是一个直径跨越最高顶线到基线的正向标准圆（`r=105`）。
- **右下缺口**：在右下侧（约 4点到 5点钟刻度）留有开口间距。
- **贯穿式尾巴（最关键调整）**：Q的尾部斜线并不是从外侧贴上去的，而是要求斜线的起点**“延伸入圆形内部，无限接近圆心但保留一点距离”**。这条干净利落的 45° 斜线从圆内中心旁起步，精准地穿过圆环右下角的物理开口，向外一直延伸，且**终止的底端必须和前方所有字母（John）的底部基线强行保持同一水平位置**，不容越界。

## 最终编排与容器对齐 (Layout & Alignment)

完成逐个字母建模后，由于 SVG 画布默认的内边距可能会造成与外部 DOM 元素的错位。我们在最后阶段通过严密的数学换算：
1. **缩放 (Scale)**：整体使用 `<g>` 标签对整个徽标执行了 `scale(0.6)` 的绝对等比例微缩。
2. **像素级对位 (Translate)**：去除了 SVG 内部左侧的物理留白，将 `J` 的最左侧边缘与下方网站主标语（"Stop overthinking" 的 S 字母）达成**左侧 0 像素误差的完美切割齐平**。
3. **顶线贴合**：通过修正 `Y` 轴 offset (-12px 的微调下移)，消除了原本的视差，让 Logo 中最高的笔画在物理像素级别上完美吻合了右侧分类导航栏（Home, Technical...）的顶部基准线。

这一切确保了最终落地的 **John Q** 并不只是几条路径的堆叠，而是经由极其精确的几何、间距公式以及物理对齐运算得到的高级定制化视觉标志。
