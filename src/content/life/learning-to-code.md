Learning to code was one of the most challenging and rewarding experiences of my life.

## The Motivation

I wanted to build things, understand how software works, and have a skill that could serve me for a lifetime.

## The Process

It took time, patience, and many frustrated moments. But every bug fixed was a victory.

## What I Learned

- Progress isn't linear
- Asking for help is okay
- Practice makes perfect
- Build things you actually use

The journey continues, but I'm proud of how far I've come.
