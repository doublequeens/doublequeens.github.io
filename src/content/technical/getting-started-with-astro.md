## What is Astro?

Astro is a modern web framework for building fast, content-focused websites. It's designed to be performant by default and highly customizable.

## Getting Started

First, install Astro:

```bash
npm create astro@latest my-blog
cd my-blog
```

Then, install Tailwind CSS:

```bash
npx astro add tailwind
```

- Customize your blog design
- Add your content
